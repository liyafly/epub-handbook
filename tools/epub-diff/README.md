# EPUB Diff (web app)

A purely browser-based EPUB diff tool. Open `index.html` in a modern browser, select two EPUB files, click Compare.

## Use

1. Run `bash scripts/fetch-vendor.sh` once to populate `assets/vendor/`.
2. Open `index.html` in Chrome / Safari / Firefox.
3. Select two `.epub` files or drag them into the pickers; click Compare.

If `file://` module loading fails, run:

```sh
cd tools/epub-diff
python3 -m http.server 8000
```

Then open `http://localhost:8000/`.

## Privacy

All processing happens in your browser. No file ever leaves your device.

## Layers

- Structure: OPF manifest, spine, nav, ncx.
- Text: block-level SHA-256 of XHTML text content.
- Style: CSS selectors and line-level diff.
- Resources: images, fonts, audio, other binary files; resource hashes are streamed through an incremental SHA-256 writer.
- Metadata: core `dc:*` fields and other metadata.

## What this tool does not do

- It does not render EPUB content or act as a reader.
- It does not run Kindle Previewer / Apple Books.
- It does not verify reading experience.
- It does not enforce the red-line policy; use `scripts/validate_text_invariance.py` for CI gates.
- It does not send data to any server.

## Dependencies (bundled after fetch)

- `@zip.js/zip.js` 2.8.26
- `@pierre/diffs` 1.2.3

Vendor JS/CSS is generated by `scripts/fetch-vendor.sh` and ignored by Git. License files are kept.
