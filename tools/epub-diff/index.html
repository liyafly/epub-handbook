About EPUB Diff

v0.1.0 · runs entirely in your browser.

This tool compares two EPUB files at the file level. It does not render books and does not verify reading experience.

For CI / red-line checks, use scripts/validate_text_invariance.py.

Diff rendering uses the built-in line renderer; vendor assets are only needed for ZIP parsing.
