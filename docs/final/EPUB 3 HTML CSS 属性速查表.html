<em>强调推荐中文显示为着重号
<strong>重要推荐语义强重要性
<cite>作品名 / 来源推荐书名、篇名、来源

<blockquote>块引用推荐长引用
<code> / <pre>代码推荐等宽字体

<dl> / <dt> / <dd>定义列表推荐术语表、参数表

<aside epub:type="footnote">弹注内容推荐同 XHTML 文件
<ruby> / <rb> / <rt> / <rp>注音推荐保留 rp 后备
